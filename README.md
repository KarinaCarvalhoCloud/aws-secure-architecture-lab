# Secure Cloud Architecture Lab – AWS

Projeto prático focado em arquitetura de soluções e segurança na AWS,
simulando um cenário real de empresa SaaS em crescimento.

## Contexto de Negócio

Uma empresa SaaS em fase de crescimento precisa estruturar seu ambiente
na AWS com foco em segurança, governança e escalabilidade. O objetivo é
garantir controle de acessos, visibilidade operacional, centralização de
logs e capacidade de resposta a incidentes, sem comprometer agilidade
dos times de desenvolvimento.

## Objetivo da Arquitetura

Projetar uma arquitetura segura em AWS que estabeleça uma base sólida
de governança, monitoramento e segurança desde o início, seguindo boas
práticas de cloud security e preparando o ambiente para crescimento
futuro e adoção de DevSecOps.

## Escopo do Projeto

### Incluído
- Estruturação de AWS Organizations
- Separação de contas por função (Management, Security, Log, Workload)
- Centralização de logs com CloudTrail
- Monitoramento e detecção de ameaças
- Controles básicos de governança e alertas

### Fora do escopo (nesta fase)
- Desenvolvimento de aplicações
- Pipelines CI/CD
- Serviços de dados avançados
- Otimizações de custo detalhadas
